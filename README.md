### Table 1: Metadata for the Mammographic Masses Dataset  
*Source: Elter & Schulz-Wendtland (n.d.)*

| Variable   | Description                                                                 | Data Type   |
|------------|-----------------------------------------------------------------------------|-------------|
| **score**  | BI-RADS assessment: 0–5                                                     | Ordinal     |
| **age**    | Patient's age in years                                                      | Continuous  |
| **shape**  | Mass shape: round=1, oval=2, lobular=3, irregular=4                         | Categorical |
| **margin** | Mass margin: circumscribed=1, microlobulated=2, obscured=3, ill-defined=4, spiculated=5 | Categorical |
| **density**| Mass density: high=1, iso=2, low=3, fat-containing=4                        | Ordinal     |
| **malignant** | benign = FALSE, malignant = TRUE                                         | Boolean     |
